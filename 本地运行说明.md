# MyAssist 本地运行说明

## Mac

双击 `启动-Mac.command`，或在终端运行：

```bash
npm run dev
```

然后打开：

```text
http://localhost:3011
```

## Windows

把整个 `MyAssist` 项目文件夹拷到 U 盘或电脑上，双击 `启动-Windows.bat`。

如果是第一次在这台 Windows 电脑运行，它会执行 `npm install` 安装依赖。需要先安装 Node.js 24 或更新版本。

## 数据存放

- macOS 旧用户数据库：`~/Library/Application Support/Leo的生活学习助手/data/leo_life_study.db`
- macOS 旧用户上传文件：`~/Library/Application Support/Leo的生活学习助手/uploads`

产品名改为 MyAssist 后仍使用原数据目录，避免重命名让已有数据看起来丢失。

搬家、备份、给朋友使用时，直接拷贝整个文件夹最稳。
